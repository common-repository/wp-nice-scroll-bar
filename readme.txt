Wp Nice Scroll Bar is a jQuery scrollbar for your wordpress website. This plugin will enable Nice Scroll Bar


===== Description =====

You can add this plugin very easily in your wordpress site. So download this plugin and your wordpress site. You can see very nice scrollbar. This plugin not any option panel so installation very easy

= Features =

*Easy make Wp Nice Scroll Bar

== Installation ==

1. Upload 'WP Nice Scroll Bar ' folder to the '/wp-content/plugins/' directory or Install as a regullar Wordpress plugin
2. You will see a scrollbar right site when you are scrolling.
